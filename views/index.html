FCC Backend Challenges

Challenge 04:Image Search Service

User stories:

    1. I can get the image URLs, alt text and page urls for a set of images relating to a given search string..
    2. I can paginate through the responses by adding a ?offset=2 parameter to the URL
    3. I can get a list of the most recently submitted search strings.

Example Links:

https://fccurlshortener.glitch.me/https://www.google.ca

Example output:

{ "url": "https://fccurlshortener.glitch.me/https://www.google.ca", "short-url": "https://fccurlshortener.glitch.me/blahblah" }

Redirect to:

https://www.google.ca
